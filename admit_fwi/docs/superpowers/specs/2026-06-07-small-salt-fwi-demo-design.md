# 小范围盐丘模型 FWI 演示设计

## 目标

在现有 `D:\ryjin\admit_fwi` 声波 RTM 工程基础上，新增一个可快速运行的小范围 SEG/Salt FWI 演示。该演示用于说明：在已有盐丘模型和 RTM 照明归一化结果之后，可以进一步用数据残差驱动速度模型更新；同时在中文说明中区分当前 RTM 的成像域照明归一化和论文中的照明优化思想。

本阶段目标是得到可复现、可展示的小规模结果，不做完整 676 x 230 网格、224 炮、4001 时间步的生产级 FWI。

## 现有基础

- 真实盐丘模型：`D:\ryjin\fd2d_pml\vel\seg676x230.bin`。
- 现有 RTM 输出目录：`D:\ryjin\admit_fwi\outputs\seg_salt_multishot_rtm_padded60_full30m_workers4`。
- 当前 RTM 照明归一化包括：
  - 震源照明归一化：`I / (L_s + epsilon)`。
  - 源检双照明归一化：`I / (sqrt(L_s L_r) + epsilon)`。
- 现有诊断显示低照明区占比约为 `0.0174`，说明完整采集条件下 RTM 结果不是主要受照明缺失控制，更明显的问题是显示压缩、低波数噪声和成像条件后处理。

## 设计原则

- 复用现有 `acoustic_rtm.py` 的速度模型读取、平滑和声波正演能力。
- 新增 FWI 演示脚本和测试，不覆盖已有 RTM 输出。
- 所有新增解释、图题、注释和 Markdown 说明统一采用中文。
- FWI 输出写入新的独立目录，避免污染 full RTM 结果。
- 小范围演示必须能在普通 CPU 环境内完成，优先保证论文展示和方法闭环。

## FWI 演示范围

小范围演示采用局部裁剪窗口：

- 从完整 SEG/Salt 模型中裁剪一个包含盐体边界或强速度变化的窗口。
- 使用平滑后的裁剪模型作为初始速度。
- 使用真实裁剪模型合成观测炮记录。
- 使用少量炮点和较短时间步进行反演。
- 输出误差下降曲线、模型对比和单炮记录对比。

初始实现默认采用固定参数，后续可通过命令行参数扩展裁剪范围、炮数、迭代次数和步长。

## FWI 更新策略

本阶段采用轻量级、展示优先的声波 FWI 近似更新：

1. 对每个炮点用当前模型正演，得到预测记录。
2. 计算预测记录与观测记录的残差，并累计均方误差。
3. 使用残差反传与正传波场的零延迟相关构造模型更新方向。
4. 对更新方向进行平滑和幅值裁剪，限制单次更新幅度。
5. 使用简单线搜索或固定小步长更新速度模型。
6. 对速度范围进行物理裁剪，避免产生非物理速度。

该策略用于演示数据残差驱动的速度更新闭环，不声明为严格工业级伴随状态 FWI 求解器。后续若需要更接近论文中的照明预处理，可在该框架上加入照明预条件梯度缩放。

## RTM 照明归一化与论文方法差异

新增中文说明需要明确以下边界：

- 当前 RTM 中的照明归一化主要是成像域幅值补偿，作用对象是偏移图像。
- 论文中的照明优化方法通常作用于采集评价、传播照明、梯度预条件、Hessian 近似或分步多参数反演。
- 因此，不能把 `I / L_s` 或 `I / sqrt(L_s L_r)` 直接等同于论文中的完整照明优化 FWI。
- 本阶段 FWI 演示用数据残差和模型更新证明“反演”与“成像归一化”的目标不同：RTM 改善反射界面成像，FWI 尝试更新速度模型。
