# RTM 成像与小范围 FWI 照明优化实验

## 方法

为进一步分析盐丘模型中 RTM 成像、照明归一化与 FWI 反演之间的关系，本文在已有二维声波 RTM 程序基础上开展小范围 FWI 数值实验。RTM 部分采用 SEG/Salt 速度模型进行多炮逆时偏移成像，并在成像域计算震源照明归一化和源检双照明归一化结果。FWI 部分从完整盐丘模型中裁剪局部窗口，使用真实裁剪模型合成观测记录，再以平滑速度模型作为初始模型，通过预测记录与观测记录之间的数据残差更新速度模型。

RTM 中的照明归一化主要作用于偏移图像。震源照明归一化可表示为：

```text
I_s^N = I / (L_s + epsilon)
```

源检双照明归一化可表示为：

```text
I_sr^N = I / (sqrt(L_s L_r) + epsilon)
```

其中，`I` 为 RTM 偏移图像，`L_s` 和 `L_r` 分别为震源波场和接收波场照明。该处理主要用于减弱照明强弱不均造成的成像振幅差异。

FWI 中的照明预条件作用于模型更新方向。本文采用正传震源波场能量构造近似照明场：

```text
L_s(x,z) = sum_t u_s(t,x,z)^2
```

并将残差反传得到的更新方向按归一化照明场进行缩放：

```text
g_pre(x,z) = g(x,z) / (L_s_norm(x,z) + epsilon)
```

因此，RTM 照明归一化和 FWI 照明预条件虽然都使用照明信息，但二者作用对象不同：前者作用于成像结果，后者作用于反演更新方向。

小范围 FWI 实验采用 `70 x 120` 的局部裁剪窗口，裁剪起点为 `z=70, x=310`，炮点位置为 `15, 60, 104`，反演迭代次数为 `3`，时间采样点数为 `360`。为分析更新尺度对反演的影响，本文依次测试固定步长、照明预条件、参数扫描、普通线搜索和自适应扩展线搜索策略。

## 实验结果

已有 full aperture RTM 诊断结果显示，盐丘模型成像结果中的低照明区占比约为 `0.0174`。该结果表明，在当前多炮采集条件下，RTM 图像并非主要受照明缺失控制，成像质量更多受到显示压缩、低波数背景和成像条件后处理影响。

固定步长 baseline FWI 将平均半均方数据残差从 `0.2963276605` 降至 `0.2936459879`，下降比例为 `0.90497%`。该结果说明小范围 FWI 反演闭环能够正常运行，但固定更新尺度下收敛幅度有限。

固定步长照明预条件 FWI 的残差下降比例为 `0.34325%`，低于 baseline。进一步进行 `epsilon` 一维扫描后，最佳参数为 `epsilon=0.2`，残差下降比例为 `0.44077%`。进行 `epsilon x max_update` 二维扫描后，最佳组合为 `epsilon=0.5, max_update=20`，残差下降比例提高到 `0.54712%`，但仍低于 baseline。

为进一步改善固定步长限制，本文引入普通线搜索策略。在线搜索候选步长 `0.25, 0.5, 1.0, 1.5, 2.0` 下，baseline FWI 残差下降比例提高到 `1.77648%`，照明预条件 FWI 提高到 `1.08500%`。两种方法在每次迭代中均选择最大候选步长 `2.0`，说明原线搜索上界偏低。

进一步采用自适应扩展线搜索策略，先测试 `0.5, 1.0, 2.0`，若最优步长位于上界则继续测试 `3.0, 4.0, 6.0, 8.0`。结果显示，baseline FWI 的残差从 `0.2963276605` 降至 `0.2774544110`，下降比例达到 `6.36905%`；照明预条件 FWI 的残差从 `0.2963276605` 降至 `0.2840638161`，下降比例为 `4.13861%`。两种方法每轮均触发扩展搜索并选择 `step_scale=8.0`。

## 讨论

实验结果表明，步长选择是当前小范围 FWI 收敛效果的关键因素。固定步长 baseline 仅获得 `0.90497%` 的残差下降，而自适应扩展线搜索 baseline 获得 `6.36905%` 的残差下降，说明原固定更新尺度明显限制了反演效果。

照明预条件在参数扫描和线搜索后均有改善，但始终未超过 baseline。固定步长照明预条件的残差下降比例为 `0.34325%`，自适应扩展线搜索后提高到 `4.13861%`，说明照明预条件确实改变了更新尺度并改善了收敛表现。然而，由于其仍低于自适应 baseline，当前轻量照明预条件不宜作为优于 baseline 的主要结论。

这一现象说明，简单的震源照明场缩放并不能完全替代更严格的 Hessian 近似或梯度预条件。若后续希望进一步突出照明优化对 FWI 的作用，应考虑频率递进、多尺度反演、改进的伴随梯度构造，或采用更接近 Hessian 对角近似的预条件方法。

## 小结

本文基于 SEG/Salt 盐丘模型完成了 RTM 成像与小范围 FWI 反演实验。RTM 照明归一化主要改善偏移图像振幅均衡，而 FWI 反演直接更新速度模型。小范围 FWI 结果表明，自适应扩展线搜索是当前最有效的改进策略，可将 baseline 残差下降比例提高到 `6.36905%`。照明预条件在调参和线搜索后同样改善，但最高下降比例为 `4.13861%`，仍未超过 baseline。因此，当前实验支持的主要结论是：步长线搜索显著提升 FWI 收敛，而轻量照明预条件可改善更新稳定性但尚不足以优于 baseline。
