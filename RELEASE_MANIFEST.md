# RTM-Only Branch Manifest

This branch is a standalone Salt RTM imaging project split from the broader salt-dome research workspace.

Included:

- RTM modeling and migration kernels.
- Single-shot and multishot SEG/Salt RTM runners.
- Imaging-condition comparison runner.
- Small-window waveform migration and direct-wave mute scan utilities.
- RTM-focused tests.
- Selected RTM result figures for GitHub front-page display.

Excluded:

- FWI update-admissibility audit framework.
- FWI optimization scripts and long-window FWI production runners.
- Large binary velocity models, wavefield checkpoints, and generated numerical outputs.
